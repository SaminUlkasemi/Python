# pydoc  
### command to generate documnentation
 - python -m pydoc -w \<filename>  
 Example: ```python -m pydoc -w main```  

### Start a server  
 - python3 -m pydoc -p \<port name>  
 Example: ```python3 -m pydoc -p 12345```
